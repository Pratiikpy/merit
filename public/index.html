How it works

Six steps. One autonomous loop.

From a question and a budget to settled, receipted payments — the agent runs the whole cycle on its own.

01
Discover

The agent surfaces candidate sources, each with a merit score and a price-per-use.

02
Escrow

USDC is locked against each source — held, not yet paid. The split stays visible.

03
Answer

It writes an answer that cites specific sources — and excludes the ones it didn't use.

04
Verify the citation

Each cited claim faces two gates: an adversarial judge that checks the source actually supports it, and a deterministic numeric verifier that re-checks every figure. Fail either and nothing pays.

05
Release / refund

Cited + verified releases payment. Uncited or unverified is refunded, with the reason shown.

06
Reputation

Merit scores update from the outcome — a portable, on-chain track record for both sides.
